The Stack

Tools I Teach & Use

Every tool on this page is free. No excuses not to start.

Core Software
🕹️
Godot Engine 4

Our game engine of choice. Godot is completely free and open-source — no royalties, no subscriptions, no strings attached. Powerful enough for serious projects and approachable enough for total beginners. Everything we build ends up here.

Primary Engine
🔵
Blender

Free, open-source, and genuinely world-class. Blender is where we model, rig, texture, and animate everything before sending it into Godot. It can feel overwhelming at first — that's exactly why we break it down step by step.

3D Creation Suite
Supporting Tools
🖌️
Krita
Free painting app for concept art and hand-painted textures.
🎨
GIMP
Free image editor for texture cleanup, normal map tweaks, and more.
🐙
GitHub
Version control for your Godot projects. Start good habits early.
📚
Godot Docs
The official docs are actually great. Bookmark them on day one.
🏃
Mixamo
Free character animations from Adobe. Works great with Godot.
🏔️
Poly Haven
100% free HDRIs, textures, and 3D models. All CC0 licensed.

💡 A note on the tools

Every tool on this page is completely free. I chose this stack specifically so that anyone — regardless of budget — can follow along with every single tutorial on this site. No trial versions, no paywalls. Just download and go.
